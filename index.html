Quick explanation of projects:
BrainExplorer is a graphics project developed to educate people about how the brain works! It's interactive and the in Pt. 2, if audio is turned on, my voice will explain how different phenomena in the brain occur (like how your brain works when you're catching a baseball). It took a while and a lot of Stack Overflow searching to code, but it was pretty fun :). The languages it uses are: JavaScript, WebGL, HTML, CSS.
